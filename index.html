03. Selected projects

Azure SOC Honeypot and Attack Map Lab

SOC lab · Azure VM · Microsoft Sentinel · KQL

Built a Windows honeypot in Azure, opened it to the internet, and captured real world attack traffic. Forwarded SecurityEvent logs to a Log Analytics workspace, connected Microsoft Sentinel, and wrote KQL queries to filter failed logons and enrich events with geo IP data using a Sentinel watchlist. Visualized attacker locations on a world map workbook.

Skills: Azure administration, NSG rules, Windows security logs, KQL, SIEM thinking.

Telco Customer Churn Analysis

Python · Classification model · SHAP explainability

End to end churn project: cleaned a telco dataset, engineered features, and trained logistic regression and random forest models to predict which customers are likely to leave. Evaluated performance with confusion matrices and classification reports, then used SHAP values and visualizations to explain which contract, payment, and usage patterns drive churn risk.

Skills: pandas, scikit learn, model evaluation, SHAP interpretation.

Operational performance: manufacturing KPI analysis

Excel · KPI dashboards · Operations

Built an Excel based KPI story for a lawn equipment manufacturer, tracking on time delivery, transmission unit cost, and retention. Cleaned and joined source tables, calculated trend metrics, and presented management ready views that highlight underperforming product lines and cost patterns.

Skills: Excel modeling, KPI design, operations analysis.

Cyclistic rider behavior analysis

R and Tableau · Bike share usage · Marketing insights

Analyzed twelve months of Chicago bike share data to compare casual and member riders. Cleaned ride logs, built summary tables for time of day and seasonality, and created Tableau visuals that support a strategy focused on converting high usage casual riders into members.

Skills: data cleaning, cohort comparison, visualization.

Bellabeat smart device usage analysis

Public Fitbit data · Engagement patterns

Explored public Fitbit data to understand how users engage with smart devices. Focused on activity intensity, daily steps, and sleep to shape product and marketing recommendations for Bellabeat on how to encourage consistent device use.

Skills: exploratory analysis, product thinking, storytelling.

Landscape database revitalization

SQL Server · schema design · ETL

Took a fragmented landscaping database and rebuilt it into an analytics ready schema. Normalized tables, created clear relationships between customers, properties, work orders, and invoices, then wrote SQL views that support reporting and margin analysis.

Skills: relational design, normalization, query optimization.

World layoffs SQL data cleaning

SQL pipeline · cleaning and standardization

Built a repeatable SQL pipeline to clean a global layoffs dataset. Standardized company names, handled missing and inconsistent dates, and created derived fields such as rolling totals and sector level summaries to support economic and workforce analysis.

Skills: data quality, window functions, transformation logic.

Operations techniques: optimization and decision models

Excel · linear models · simulation

Applied an operations research toolkit across several business scenarios including queues, payoff and regret tables, Monte Carlo simulations, linear programs, decision trees, and binary integer models. Focused on translating real constraints into models that leadership can use to pick a clear strategy.
