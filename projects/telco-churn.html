Telco Customer Churn Analysis

Built an end-to-end machine learning workflow to predict customer churn, identify the strongest churn indicators, and create data-driven recommendations for telecom retention strategy.

Logistic Regression · Random Forest SHAP Model Explainability

Project Narrative

This project applies machine learning to a telecommunications customer dataset to predict which subscribers are most likely to churn. After preparing the data, I trained logistic regression and random forest models, evaluated their performance, and used SHAP values to understand which factors influence churn the most.

The workflow produced a reliable predictive model and clear insights that inform retention strategy—highlighting key customer segments and risk behaviors.

What I Built

  • Loaded and explored the Telco churn dataset.
  • Cleaned and converted problematic variables (including TotalCharges).
  • Performed feature engineering and one-hot encoding for all categorical fields.
  • Built models using logistic regression and random forest.
  • Evaluated performance using accuracy, recall, precision, and ROC-AUC.
  • Generated SHAP values to understand the strongest churn drivers.
  • Developed insights and recommendations for customer retention.

Dataset Overview

The dataset contains 7,000+ customer records with fields covering demographics, billing, service usage, and contract information. The target variable is Churn (Yes/No).

  • Demographics: age, partner status, dependents
  • Services: tech support, internet type, streaming
  • Billing: monthly charges, payment method, tenure
  • Contract types: month-to-month or annual

Key Churn Drivers

  • Month-to-month contracts were the strongest churn predictor.
  • Customers with high monthly charges churned at significantly higher rates.
  • Limited support features (no tech support or security add-ons) increased churn likelihood.
  • Electronic check billing was consistently correlated with higher churn.
  • Fiber-optic users showed lower churn, suggesting service quality matters.
