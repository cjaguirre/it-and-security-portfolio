What I built

An Excel workbook with separate tabs for each KPI area, plus charts and statistical tests to validate whether observed differences are real or just noise. The work mirrors a real operations analyst workflow — organize the data, build the KPIs, run tests at a defined confidence level, then write up clear recommendations for leadership.

Key outcomes

  • • On-time delivery improvement confirmed statistically (proportion test)
  • • Process A identified as lowest-cost option via ANOVA
  • • Retention patterns segmented by gender and locality with pivot tables
  • • Seasonal delivery trends surfaced for staffing and inventory planning
  • • 8-page Word report summarizing findings and recommendations

The Workflow

Step 1

Data Import

Loaded delivery, cost, and retention data into structured Excel tabs with labeled ranges.

Step 2

Cleaning

Formatted ranges, checked for errors, and validated that all 2014–2018 values aligned across tabs.

Step 3

KPI Construction

Built delivery percent, unit cost, and retention metrics using COUNTIFS, SUMIFS, and pivot tables.

Step 4

Statistical Testing

Applied a proportion test for delivery and one-way ANOVA for costs using the Data Analysis ToolPak.

Step 5

Reporting

Summarized findings and recommendations in an 8-page Word report for a management audience.

Analysis Breakdown

On-Time Delivery — Did It Actually Improve?

The question wasn't just whether 2018 delivery rates looked better than 2014 — it was whether that difference is statistically meaningful or just normal variation. A line chart shows the visual trend, then a one-sample proportion test at 95% confidence gives the formal answer.

What the chart shows

  • • Monthly delivery rates for 2014 and 2018 plotted side by side
  • • 2018 consistently outperforms 2014 across most months
  • • Seasonal dips visible in both years — useful for staffing and inventory planning

Monthly delivery comparison for 2014 and 2018 — seasonal patterns and overall improvement visible

The proportion test

  • • One-sample proportion test at α = 0.05
  • • Null hypothesis: 2018 delivery rate is no better than 2014
  • • Result: null rejected — the improvement is statistically significant

Proportion test results — 2018 improvement confirmed at 95% confidence

Transmission Cost — Which Process Is Cheapest?

Three manufacturing processes were evaluated: the current process, Process A, and Process B. A bar chart gives the visual comparison, and a one-way ANOVA confirms whether the cost differences are statistically real — not just sample variation.

Visual comparison

  • • Unit costs plotted for all three processes
  • • Process A visually sits at the lower end of the cost range
  • • Variability also visible — Process B shows wider spread

Unit transmission cost across all three processes — Process A consistently lower

ANOVA results

  • • One-way ANOVA via Excel Data Analysis ToolPak
  • • F statistic, critical F, and p-value all reported
  • • Result: statistically significant cost differences confirmed
  • • Process A recommended as the most cost-efficient option

ANOVA output — statistically significant cost differences across the three processes

Employee Retention — Who's Staying?

Retention was analyzed by gender and locality using pivot tables. The goal was to identify whether certain demographic segments show meaningfully different retention patterns — and which groups might benefit from more targeted HR attention.

What the analysis shows

  • • Pivot table segments retention by gender and locality combinations
  • • Bar chart highlights stronger and weaker retention groups at a glance
  • • Differences present but generally modest — overall workforce is stable
  • • Specific segments worth closer HR monitoring identified

Key Excel Formulas

Behind the charts and statistical outputs, the workbook relies on standard Excel functions to calculate KPIs, filter by date and demographic, and feed the ToolPak tests with clean inputs.

Conditional count — delivery

=COUNTIFS(A:A, "Complete",
          B:B, ">"&DATE(2018,1,1))

Counts completed deliveries after January 1 2018 — used to filter valid events for the proportion test.

Retention rate by segment

=SUMIFS(Retained, Gender, "F")
 / SUMIFS(Employed, Gender, "F")

Calculates retention rate for female employees — divides retained count by total employed in the same segment.

Supporting the statistical tests

  • • Proportion calculations built from basic division and sample size checks
  • • Data validation ranges ensure only valid periods feed the ToolPak procedures
  • • Helper columns standardize inputs before running ANOVA
