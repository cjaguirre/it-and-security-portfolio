Operational Performance: Manufacturing KPI Analysis

Designed a full operational KPI dashboard for a lawn equipment manufacturer, combining cleaned datasets, calculated metrics, and Excel visualizations to uncover patterns in delivery performance, unit cost, product margins, and retention.

Excel • KPI Dashboards Operational Performance

Project Narrative

The goal was to evaluate production performance using real operational data. Multiple files were cleaned, standardized, and joined into a structured data model. I then created calculated KPI fields and built an interactive dashboard that summarizes trends for leadership.

The final product provides a clear, consolidated view of efficiency, cost behavior, labor stability, and product-line contribution.

What I Built

  • Cleaned raw operational files and corrected inconsistent fields.
  • Created calculated KPIs including delivery %, retention, unit cost, and defect rate.
  • Used pivot tables, relationships, and slicers for drill-down analysis.
  • Designed an executive dashboard summarizing key operational trends.
  • Highlighted inefficiencies and product-line performance issues.

Data & Methodology

The analysis started with raw CSV and Excel files extracted from operational logs. After cleaning and combining them, I calculated KPIs and built an Excel dashboard to visualize month-to-month performance.

  • Standardized inconsistent headers and formats.
  • Removed duplicate rows and corrected missing values.
  • Built calculated fields for performance KPIs.
  • Created pivot models with slicers for flexible analysis.

Key KPIs Calculated

  • On-Time Delivery % — share of orders shipped by or before the scheduled date.
  • Transmission Unit Cost — total cost per produced unit.
  • Retention Rate — monthly production employee retention.
  • Defect Rate — defective units divided by total output.
  • Product Line Contribution — margin analysis to identify high-value items.

Sample Excel Logic

# On-Time Delivery %
=IF([@[Ship Date]] <= [@[Due Date]], 1, 0)

# Retention Rate
=([@[Employees Start]] - [@[Employees Separated]]) / [@[Employees Start]]

# Transmission Unit Cost
=[@[Direct Labor]] + [@[Parts Cost]] + [@[Overhead]] + [@[Rework Cost]]
