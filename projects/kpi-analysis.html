Project Narrative

This senior-level operations analytics project evaluated critical performance KPIs for a lawn equipment manufacturer. Through structured Excel models, KPI dashboards, hypothesis testing, and a formal written report, I identified the drivers behind delivery delays, rising unit costs, and uneven employee retention.

Deliverables included a multi-sheet Excel analysis workbook and an 8-page report summarizing KPI findings, test results, and strategic recommendations.

This project demonstrates a complete business intelligence workflow: from raw data, to statistical validation, and finally to generating actionable recommendations for management.

2. On-Time Delivery Analysis

Proportion Test · Trend Analysis

This part of the analysis evaluates whether the company’s on-time delivery rate in 2018 meaningfully improved compared to 2014. A one-sample proportion test confirmed that the 2018 delivery rate was significantly higher at the 95 percent confidence level, showing that operational adjustments made over the four-year period had a measurable impact.

The first screenshot provides a visual comparison of monthly delivery performance for both years. This chart makes it easy to spot seasonal dips, recovery periods, and the overall upward improvement in delivery consistency. Seeing both lines together gives a clear sense of how 2018 outperformed 2014 across most months.

Monthly delivery comparison for 2014 and 2018, showing seasonal patterns and overall improvement.

The second screenshot displays the statistical portion of the analysis: the one-sample proportion test used to verify that the improvement in delivery percentage was not due to chance. The output includes the test statistic, critical values, p-value, and the final decision to reject the null hypothesis. This confirms that the observed improvement in 2018 is statistically significant.

Proportion test results verifying that the improvement in on-time delivery during 2018 is statistically significant.

3. Transmission Cost Analysis

One-way ANOVA · Process Comparison

This analysis focuses on whether unit transmission costs differ meaningfully across the three manufacturing processes: the current process, Process A, and Process B. A one-way ANOVA was performed using Excel’s Data Analysis ToolPak to determine if these cost differences were statistically significant. Results showed that Process A had the lowest mean cost, making it the most cost-efficient option.

The first screenshot displays the unit cost comparison chart. Each bar represents a transmission unit cost observation for one of the three processes. This chart provides an immediate visual sense of cost variability — the spread, the height differences, and how consistently each process performs. Even before testing, it’s clear that Process A trends lower than the others.

Unit transmission cost comparison across the current process, Process A, and Process B.

The second screenshot shows the full ANOVA output used to statistically validate the cost differences. It includes the summary statistics for each process, the calculated F-statistic, the critical F-value, and the resulting p-value. Because the p-value is below the significance threshold, the test confirms that at least one process has a cost structure meaningfully different from the others — supporting the conclusion that Process A is the most cost-efficient option.

ANOVA test confirming statistically significant cost differences across processes.

5. Key Formula Examples

Excel functions for data modeling and testing

Although ANOVA and other complex tests used the Data Analysis ToolPak, core KPI modeling relied on standard Excel statistical and logical functions.

  • Proportion Test: Z-score calculation using NORMSINV and SQRT.
  • Retention Metrics: Built with COUNTIFS, SUMIFS, and demographic filters.

Conditional Count 

=COUNTIFS(A:A, "Complete", B:B, ">"&DATE(2018,1,1))

Counts all entries marked "Complete" that occurred after January 1, 2018. Used for filtering valid events across a specific time range.

Retention Percentage 

=SUMIFS(Retained, Gender, "F") / SUMIFS(Employed, Gender, "F")

Calculates the retention rate for female employees by dividing those retained by the total number employed within the same demographic.
