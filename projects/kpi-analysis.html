Operations Management Analysis

A complete operations analytics project built in Microsoft Excel and Word to evaluate manufacturing KPIs, analyze delivery performance, compare transmission costs, and assess employee retention patterns. The analysis provides data-backed recommendations to support operational decision-making.

Excel Dashboards · KPI Modeling Statistical Analysis · ANOVA · Proportion Tests

Project Narrative

This senior-level operations analytics project evaluated critical performance KPIs for a lawn equipment manufacturer. Through structured Excel models, KPI dashboards, hypothesis testing, and a formal written report, I identified the drivers behind delivery delays, rising unit costs, and uneven employee retention.

Deliverables included a multi-sheet Excel analysis workbook and an 8-page report summarizing KPI findings, test results, and strategic recommendations.

This project demonstrates a complete business intelligence workflow: from raw data, to statistical validation, and finally to generating actionable recommendations for management.

What I Built

  • Excel workbook for KPI analysis and statistical testing.
  • On-time delivery, transmission cost, and retention dashboards.
  • One-sample proportion testing for delivery improvements.
  • ANOVA testing for cost differences across processes.
  • Demographic retention analysis (gender, locality).
  • A structured written report summarizing all insights.

2. On-Time Delivery Analysis

Proportion test · Trend analysis

Delivery performance improved significantly in 2018 compared to 2014. A one-sample proportion test confirmed the improvement was statistically significant at a 95 percent confidence level, validating the operational changes made across four years.

  • Clear seasonal dips and peaks in delivery rates.
  • Annual delivery percentage showed a strong upward trend.

3. Transmission Cost Analysis

One-way ANOVA · Process comparison

ANOVA was used to determine if the mean unit cost differed significantly across three manufacturing processes (A, B, C). Results confirmed significant differences, with Process A emerging as the most cost-efficient option.

  • Used Excel's Data Analysis ToolPak for ANOVA.
  • Calculated F-statistic and P-value to determine significance.

5. Key Formula Examples

Excel functions for data modeling and testing

Although ANOVA and other complex tests used the Data Analysis ToolPak, core KPI modeling relied on standard Excel statistical and logical functions.

  • Proportion Test: Z-score calculation using NORMSINV and SQRT.
  • Retention Metrics: Built with COUNTIF, SUMIFS, and demographic filters.

Conditional Count

=COUNTIFS(A:A, "Complete", B:B, ">"&DATE(2018,1,1))

Retention Percentage

=SUMIFS(Retained, Gender, "F") / SUMIFS(Employed, Gender, "F")
