Operational Performance Analysis for a Manufacturing Company

A full operational analytics project built in Microsoft Excel and Word to diagnose performance issues in a lawn equipment manufacturing company. The project analyzes on-time delivery KPIs, transmission cost efficiency, and employee retention to provide data-driven recommendations that support better operational decision-making.

Excel Dashboards · KPI Modeling Statistical Analysis · ANOVA · Proportion Tests

Project Narrative

This senior-level analytics project evaluated key operational KPIs for a lawn equipment manufacturer. Using structured Excel workbooks, statistical testing, and business-focused reporting, I identified performance trends, cost inefficiencies, and retention issues that directly impact productivity and profitability.

The deliverables included a multi-tab Excel analysis workbook and an 8-page report summarizing KPI trends, test results, and recommendations.

What I Built

  • Multi-tab Excel workbook for KPI analysis.
  • On-time delivery, transmission cost, and employee retention dashboards.
  • One-sample proportion test for delivery improvement (2014 → 2018).
  • ANOVA testing to compare transmission process costs.
  • Retention breakdowns by gender and locality.
  • Full Word-based report summarizing results and insights.

Workflow Overview

Step 1

Excel workbooks imported with raw delivery, cost, and retention data.

Reporting

2. On-Time Delivery Analysis

One-sample proportion test · Delivery trend visualization

Delivery performance was analyzed from 2014 to 2018. A one-sample proportion test showed a statistically meaningful improvement in 2018 compared to baseline 2014 delivery rates.

  • Monthly trend analysis revealed seasonality patterns.
  • Delivery performance rose steadily across the time period.

[ Screenshot: BUS 310 CSP OTD Percentage ]

3. Transmission Cost Analysis

One-way ANOVA · Cost comparison across processes

A one-way ANOVA test was performed to compare transmission process costs. Although differences were not large, Process A emerged as a slightly more cost-effective alternative.

  • Evaluated cost components across several processes.
  • ANOVA showed marginal variances but actionable insights.

[ Screenshot: BUS 310 CSP UTC for Processes ]

4. Employee Retention Analysis

Gender · Locality · Retention trends

Employee retention was evaluated using demographic breakdowns. Local employees and male employees showed slightly higher retention, informing targeted retention strategies.

Key Outcomes

  • Process A identified as slightly more cost-effective.
  • Retention patterns revealed demographic risk factors.
  • Excel workbook and report provide reusable templates for operational analysis.
