BellaBeat Smart Device Usage Analysis

Public Fitbit Dataset • Engagement Patterns • Product Insights

This analysis explores public Fitbit data to understand how users engage with smart devices across daily activities, movement patterns, and sleep behavior. The goal was to identify trends that could inform BellaBeat’s product and marketing strategy, helping the company promote more consistent device usage among its target audience.

Using a combination of exploratory analysis and descriptive statistics, I analyzed activity intensity, total steps, calorie burn, sleep records, and daily engagement metrics. These insights show how habits differ between high-engagement and low-engagement users, and what lifestyle patterns correlate with consistent device interaction.

Dataset & Tools

  • 12,000+ rows of daily Fitbit usage data
  • Activity minutes broken into sedentary, lightly active, fairly active, and very active
  • Sleep logs showing total time asleep, time in bed, and sleep efficiency
  • Daily step counts and calories burned
  • Tools used: Python, pandas, seaborn, Matplotlib

Key Findings

  • High-engagement users averaged significantly more daily steps (10k+), correlating with stronger device usage habits.
  • Sleep patterns were a consistent marker of user segmentation: users with 6–7 hours of sleep showed the highest day-to-day device engagement.
  • Sedentary minutes were the strongest negative indicator of device engagement.
  • Calories burned and active minutes tracked closely with step count tiers, identifying opportunities for BellaBeat to promote activity streaks.

Sample Python Code

import pandas as pd

df = pd.read_csv("dailyActivity.csv")

# Summaries
df_summary = df.groupby("Id")[["TotalSteps", "SedentaryMinutes", "Calories"]].mean()

# Categorize users by activity
df['activity_level'] = pd.cut(
    df['TotalSteps'],
    bins=[0, 5000, 10000, 20000],
    labels=['Low activity', 'Moderate activity', 'High activity']

Strategic Insights for BellaBeat

  • Marketing campaigns should encourage **daily step streaks** to reinforce consistent activity patterns.
  • Users respond well to **moderate sleep improvement goals** rather than highly aggressive targets.
  • Device notifications could focus on breaking long sedentary periods, using timely nudges.
  • Retention efforts should target users with declining step counts—an early sign of disengagement.
