Bellabeat Smart Device Usage Analysis

Analysis of activity, sleep, and wellness behavior from Fitbit data to uncover trends, improve Bellabeat product engagement, and identify marketing opportunities based on real user habits.

Activity & Sleep Patterns Behavioral Insights

Project Narrative

This capstone analysis explores daily movement, sleep, and activity trends to understand how Bellabeat users interact with wellness devices. Usage signals such as total steps, active minutes, calories burned, and sleep duration reveal strong relationships between behavior and device engagement.

These insights help Bellabeat refine its marketing and product strategy by identifying when and how users rely on their smart devices the most.

What I Built

  • Imported, cleaned, and merged Fitbit activity, sleep, and calorie datasets.
  • Calculated new metrics such as active minutes and weekday behavior.
  • Analyzed trends across steps, active minutes, sleep duration, and calories.
  • Created visuals highlighting weekday patterns and wellness habits.
  • Developed insights to support Bellabeat’s strategic marketing goals.

Dataset & Tools

The dataset includes 30 days of Fitbit data from 30 users, covering daily steps, calories burned, active minutes, and sleep metrics. Analysis was completed using R, tidyverse, ggplot2, and Tableau for visualization.

  • Fitbit activity, sleep, and calorie datasets
  • 30 days of user behavior
  • Daily movement intensity + time asleep
  • R, tidyverse, ggplot2, Tableau

Key Findings

  • Users walked the most on weekdays, with peak steps mid-week.
  • Sleep duration averaged 6–7.5 hours—below the recommended 8 hours.
  • Calories burned strongly correlated with active minutes.
  • Weekends showed the largest drop-off in wellness behavior.

Sample R Code

# Import Fitbit data
daily_activity <- read_csv("dailyActivity_merged.csv") %>%
  clean_names()

daily_sleep <- read_csv("sleepDay_merged.csv") %>%
  clean_names()

# Merge
merged <- daily_activity %>%
  inner_join(daily_sleep, by = c("id", "activity_date" = "sleep_day"))

# Create features
cleaned <- merged %>%
  mutate(
    day_of_week = wday(activity_date, label = TRUE),
    active_minutes_total = very_active_minutes +
                           fairly_active_minutes +
                           lightly_active_minutes
  )

glimpse(cleaned)
