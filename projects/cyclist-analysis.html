Cyclistic Rider Behavior Analysis

R & Tableau • Bike-share usage • Rider segmentation • Marketing insights

This project analyzes twelve months of Chicago’s Cyclistic bike-share data to identify key differences between casual riders and annual members. The goal was to understand how riding habits vary across seasons, weekdays, time of day, and trip purposes — insights used to support Cyclistic’s marketing strategy to convert casual riders into members.

Using R for data wrangling and Tableau for visualization, this analysis reveals the behavioral patterns and peak usage windows that distinguish the two rider groups, enabling clear recommendations for membership growth and targeted promotions.

Dataset & Tools

  • 12 months of Cyclistic trip data (over 5 million rides)
  • Trip durations, rideable types, start/end times, station locations
  • R for merging, cleaning, and summarizing datasets
  • Tableau for interactive dashboards and visual exploration

Key Findings

  • Casual riders peak on weekends, especially afternoons during warmer seasons.
  • Members ride more consistently year-round, particularly during weekday commute hours.
  • Electric bikes are more popular with casual riders due to convenience and leisure use.
  • Trip durations for casual riders are significantly longer, often tied to recreational routes.
  • Members favor shorter, utilitarian trips aligned with commuting or errands.

Sample R Code

library(tidyverse)

# Load and combine monthly files
files <- list.files(pattern = "*.csv")
all_trips <- files %>% 
  map_df(~ read_csv(.))

# Clean durations and remove bad data
clean_trips <- all_trips %>% 
  mutate(ride_length = ended_at - started_at,
         day_of_week = weekdays(started_at)) %>%
  filter(ride_length > 0)

# Summary comparison
clean_trips %>% 
  group_by(member_casual) %>% 
  summarise(
    avg_duration = mean(ride_length),
    total_rides = n()
  )

Strategic Insights for Cyclistic

  • Promotions should target weekend casual riders with “Ride More This Summer” membership discounts.
  • Introduce commuter-focused member perks such as guaranteed bike availability at key stations.
  • Highlight the cost savings of frequent short rides to appeal to casual riders with long trip histories.
  • Push app notifications during peak conversion windows (warm months, weekend afternoons).
  • Create routes and challenges that reward consistent usage, bridging casual → member behavior.

Skills Demonstrated

Data cleaning, R wrangling, cohort comparison, Tableau dashboarding, behavioral analytics, strategic insights.

Summary

This project demonstrates how large behavioral datasets can shape marketing and retention strategies. By identifying exactly how and when riders use Cyclistic bikes — and what differentiates casual users from loyal members — the company can create targeted campaigns aimed at boosting membership and driving long-term ridership.
