World Layoffs — SQL Data Cleaning

SQL • Data Cleaning • Exploratory Analysis • ETL Concepts • Standardization • Data Quality

This project focuses on cleaning and preparing a global layoffs dataset for analytics. Using SQL, I corrected structural issues, standardized inconsistent fields, handled missing values, removed duplicates, and transformed the dataset into an analysis-ready format.

The final cleaned dataset supports reporting on global workforce trends, industry impact, and corporate downsizing patterns during major economic shifts.

Dataset Overview

The original dataset included records from companies around the world with fields such as:

  • Company Name
  • Country
  • Industry
  • Location
  • Total Employees
  • Layoff Count
  • Percentage Laid Off
  • Layoff Date

Raw data contained numerous inconsistencies, making cleaning essential before analysis.

Key Data Cleaning Steps

  • Removed duplicate entries using window functions.
  • Standardized inconsistent company names (e.g., "Google LLC" vs "Google").
  • Filled missing industry values using company reference groups.
  • Converted date strings into SQL date format.
  • Trimmed whitespace and fixed casing issues.
  • Removed invalid records with incomplete or contradictory data.
-- Example: Remove duplicates
WITH ranked AS (
    SELECT *,
    ROW_NUMBER() OVER (
        PARTITION BY company, location, date
        ORDER BY layoffs DESC
    ) AS rn
    FROM layoffs_raw
)
DELETE FROM ranked WHERE rn > 1;
        
-- Standardizing industry by pulling most common value
UPDATE layoffs
SET industry = (
    SELECT mode() WITHIN GROUP (ORDER BY industry)
    FROM layoffs AS l2
    WHERE l2.company = layoffs.company
)
WHERE industry IS NULL;

Data Transformations

After the cleaning phase, I added several derived fields to support trend analysis:

  • Year extracted from the layoff date
  • Quarter derived for period trend reporting
  • Categorized industries into standardized business groups
  • Country normalization for consistent geography analytics
ALTER TABLE layoffs_clean
ADD COLUMN year INT;

UPDATE layoffs_clean
SET year = EXTRACT(YEAR FROM layoff_date);

Insights Enabled by Cleaning

  • Identification of industries most impacted (Tech, Finance, Retail)

  • Time-series trends showing spikes during recessions
